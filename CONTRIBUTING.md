# Contributing

Contributions should follow the same standard this project promotes.

For every pull request:

1. Generate changes through an agent conversation.
2. Publish the conversation transcript or link to it from the pull request.
3. Do not manually edit generated files outside the agent workflow.
4. If something is wrong, ask the agent to fix it in a follow-up conversation.

If a contribution cannot meet this standard, say so plainly. The project can still discuss it, but it should not be represented as pure vibe coding.
